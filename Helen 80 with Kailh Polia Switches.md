# Helen 80 開箱
  
  
如題  
這次要開的是Helen 80  
其實已經收到一段時間了  
但一直沒有動工  
就趁連假迅速組裝一波  
  
順道買了凱華新出的杜若軸(Kailh Polia Switch)  
本來就喜歡段落軸  
加上顏色是淺紫色就更加分了  
鍵帽的部分則選擇了太豪  
  
![](https://i.imgur.com/qhxpIxa.jpg)  
登愣~  
三個盒子分別裝了軸.鍵帽.Helen80本體  
箱子上就印著Playkeyboard x KBParadise以及HELEN 80  
簡簡單單 一目了然  
  
  
![](https://i.imgur.com/eVsUr3X.jpg)  
打開盒子  
裡面裝有  
- Helen 80 套件 一組  
- Helen銘牌貼紙 一組  
- Type C線 一條  
- 定位板衛星軸 一組  
- 套件螺絲一組(小螺絲有貼心的多給一份)  
- 拔鍵器一枚  
  
其中還有兩個吸音綿  
一塊是放在底殼  
一塊是夾在PCB和定位板之間的  
用來填滿空隙   
減少敲擊的時候帶來的空腔音  
  
由於PCB是熱插拔  
連led指示燈都是呢  
組裝的過程蠻簡單的  
衛星軸處理後裝上  
軸的位置放對壓好  
led燈插上  
該鎖的螺絲鎖一鎖  
上蓋蓋上卡好  
就差不多了  
  
組裝的過程忘了紀錄  
所以...
